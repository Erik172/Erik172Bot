Bot like and RT

Este bot usa el flujo de Tweepy para buscar activamente tweets que contengan ciertas palabras clave. por cada tweet, si usted no es el autor del tweet, le gustará el tweet y luego lo retuiteará.

Puede utilizar este bot para alimentar su cuenta con contenido que sea relevante para sus intereses.

A continuación, puede ver el código fuente completo de este bot. Utilice una secuencia para filtrar los tweets que contienen las palabras "Platzi" o "Python". A cada tweet en la corriente se le da me gusta y se retuitea:

la función principal usa create_api() del módulo de configuración para crear un objeto API Tweepy.

Se crea un flujo de Tweepy para filtrar tweets que están en el idioma español e incluyen algunos de los palabras clave especificadas en el argumento de la función principal, "Platzi" o "Python" en este caso.

El on_status() de FavRetweetListener procesa los tweets de la secuencia. Este método recibe un objeto de estado y usa favorite() y retweet() para marcar el tweet como Me gusta y retweet.

Para evitar retuitear y dar me gusta a los tweets que son respuestas a otros tweets, on_status() usa un if para vea si in_reply_to_status_id no es None. Además, el código comprueba si usted es el autor del tweet para evitar retuitear y dar me gusta a su propio contenido.


Following Followers

Este bot obtiene su lista de seguidores de Twitter cada minuto y luego la recorre para seguir a cada usuario que aún no está siguiendo.

follow_followers() utiliza un cursor Tweepy y el método Tweepy API followers() para obtener su lista de seguidores. Esta lista contiene un modelo de usuario de Tweepy para cada usuario que te sigue.

Luego, el bot recorre la lista y usa el seguimiento para verificar si ya está siguiendo a cada usuario. Los usuarios que aún no están siendo seguidos son seguidos usando follow().
